# Política de Privacidade do Soroban Kids

Última atualização: 5 de maio de 2026

O Soroban Kids é um aplicativo educativo criado para ensinar o uso do soroban de forma visual, interativa e divertida.

Esta política explica como o app trata informações no aparelho.

## Resumo

O Soroban Kids funciona completamente offline.

O app:

- não coleta dados pessoais;
- não compartilha dados com terceiros;
- não exige cadastro;
- não possui conta online;
- não usa internet;
- não usa analytics;
- não exibe anúncios;
- não possui compras dentro do app;
- não envia dados para servidores.

## Dados salvos no aparelho

O app pode salvar localmente, apenas no próprio aparelho:

- nome do perfil criado pelo usuário;
- avatar escolhido;
- progresso nos níveis;
- recompensas desbloqueadas;
- preferências de tema;
- preferência de quantidade de hastes do soroban;
- preferências de música e efeitos sonoros;
- configuração do modo Anzan;
- configuração do desafio de tempo;
- recordes locais do desafio de tempo.

Essas informações ficam somente no aparelho e não são transmitidas para nenhum servidor.

## Crianças

O Soroban Kids foi criado para apoiar o aprendizado de crianças, com foco em matemática e uso do soroban.

O app não solicita e-mail, telefone, localização, câmera, microfone, contatos ou qualquer dado sensível.

## Internet

O Soroban Kids não precisa de internet para funcionar.

No Android, a permissão de internet é bloqueada na configuração do app.

## Anúncios e compras

O app não possui anúncios, compras dentro do app, assinaturas ou conteúdo pago.

## Compartilhamento de dados

O Soroban Kids não compartilha dados com terceiros.

## Exclusão de dados

Como os dados ficam apenas no aparelho, o usuário pode removê-los:

- excluindo os perfis dentro do próprio app, quando disponível;
- limpando os dados do app nas configurações do sistema;
- desinstalando o app.
